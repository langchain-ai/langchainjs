## Summary

Completes the removal of the `@langchain/community` meta-package and `libs/community/` directory, while preserving the important provider integrations that lived there by relocating them to `libs/providers/`. Builds on top of `cb/remove-community` which already extracted fireworks, together-ai, perplexity, IBM, neo4j, and pgvector.

## Changes

### Packages relocated from `libs/community/` → `libs/providers/`

- `@langchain/cohere`
- `@langchain/exa`
- `@langchain/pinecone`
- `@langchain/qdrant`
- `@langchain/tavily`
- `@langchain/weaviate`

- `@langchain/azure-cosmosdb`
